# 🚀 Space Shooter Game

A 2D arcade-style space shooter game built using Python and Pygame.  
The player controls a spaceship, shoots lasers, and destroys incoming enemy ships while avoiding collisions.

## 🎮 Features

- Player controlled spaceship
- Multiple enemy ships
- Laser shooting mechanics
- Collision detection
- Explosion animations
- Background music and sound effects
- Score and health tracking

## 🛠 Technologies Used

- Python
- Pygame
